# find_book

### 실행 순서

**서버 실행**

```
npm run schema
```

인터페이스를 변경하지 않는 한 최초 한 번만 실행하면 된다.
인터페이스를 변경할 시 반드시 실행해야 한다.

```
npm run dev
```

**테스트**

```
npm run test
```

### 사용 기술
